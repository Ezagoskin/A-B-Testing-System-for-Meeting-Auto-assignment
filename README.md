### Repository with code for Software Project A/B Testing System for Meeting Auto-assignment 
The switchback directory of the repository contains the switchback service modules, including the scheduling API, the settings update scheduler, logging utilities, request validation, and the internal API client. 

The mde_functions.py contains functions for regional tests, switchback tests, CUPED, and CUMPED. CC_delivery_MDE.ipynb shows an example of using these functions for estimating MDE for CC delivery metrics. Data for CC delivery MDE.sql illustrates data preparation for CC delivery MDE simulations. 

Region selection.sql provides scripts for choosing regions suitable for A/B testing.
